USAGE

INPUT

model â a COBRA model

OPTIONAL INPUTS

  • findNCgaps â find no consupmption gaps as well as no production gaps (default false)

  • verbFlag â verbose flag (default false)

OUTPUTS

  • allGaps â all gaps found by GapFind

  • rootGaps â all root no production (and consumption) gaps

  • downstreamGaps â all downstream gaps

Based on Kumar, V. et al. BMC Bioinformatics. 2007 Jun 20;8:212.
