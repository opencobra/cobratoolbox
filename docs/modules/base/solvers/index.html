LP - linear optimisation

\[ \begin{align}\begin{aligned}\begin{split} max/min ~& c^T x + 0.5 x^T F x \\ s.t. ~& [S, E; C, D] x + r <=> b ~~~~~~~~~~~:y \\ ~& lb \leq x + u \leq ub~~~~:w\end{split}\\QP - quadratic optimisation\end{aligned}\end{align} \]

USAGE

INPUTS

  • model1 â The first COBRA model

  • model2 â The second COBRA model, where both models have mandatory fields:

    • S - Stoichiometric matrix

    • b - Right hand side = 0

    • c - Objective coefficients

    • lb - Lower bounds

    • ub - Upper bounds

  • osenseStr â Maximize (âmaxâ)/minimize (âminâ) (Default = âmaxâ)

  • minFluxFlag â Minimize the absolute value of fluxes in the optimal MOMA solution (Default = false)

  • solution1 â Solution for the 1st model

  • solution2 â Solution for the 2nd model

  • totalFluxDiff â 1-norm of the difference between the flux vectors sum|v1-v2|

lb1 <= v1 <= ub1

f2 = max/min c2âv2 subject to S2*v2 = b2

s.t. S1*v1 = b1

c1âv1 = f1 lb1 <= v1 <= ub1 S2*v2 = b2 c2âv2 = f2 lb2 <= v2 <= ub2

Finally optionally minimizes the 1-norm of the flux vectors

  • 3 - Almost optimal solution
